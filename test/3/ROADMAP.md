# ROADMAP.md

## ✅ Выполнено (v1.0.0)

- [x] MVVM архитектура (Model, ViewModel, View)
- [x] Базовый калькулятор (+, -, *, /, скобки, десятичные)
- [x] Интерфейс в стиле Excel (сетка кнопок + строка формул)
- [x] Голосовой ввод через Vosk (русский язык)
- [x] Парсер чисел до 100 прописью
- [x] Потокобезопасность голосового ввода
- [x] Безопасные вычисления через AST (без eval)
- [x] Обработка ошибок (деление на ноль, синтаксис, пустой ввод)
- [x] Документация (README, CHANGELOG, FAQ, ROADMAP)

## 🤝 Как внести вклад

1. Выберите задачу из плана (приоритет высокий/средний)
2. Форкните репозиторий
3. Создайте ветку: `feature/название-фичи`
4. После реализации откройте Pull Request
5. Дождитесь ревью
