# CHANGELOG.md

## [v1.0.0] - 2026-04-24

### Added
- **MVVM архитектура**: строгое разделение Model, ViewModel, View
- **CalculatorModel**: безопасное вычисление выражений через AST с валидацией узлов
- **CalculatorViewModel**: обработка команд, управление состоянием, сигналы для View
- **CalculatorView**: интерфейс в стиле Excel (сетка кнопок 5x4 + строка формул)
- **VoiceInputHandler**: интеграция с Vosk, захват звука через pyaudio, потокобезопасность
- **CommandParser**: преобразование русской речи в математические выражения
  - Поддержка чисел до 100 (1-19, десятки, составные)
  - Операторы: плюс, минус, умножить, разделить
  - Скобки: открыть/закрыть скобку
  - Десятичные дроби: точка/запятая
- **Голосовое управление**:
  - Кнопка запуска/остановки распознавания
  - Визуальная индикация (кнопка становится красной)
  - Автоматическая остановка после 2 секунд тишины
- **Обработка ошибок**:
  - Деление на ноль → сообщение в строке формул
  - Нераспознанная команда → "Не распознано"
  - Пустой ввод → игнорирование
  - Синтаксическая ошибка → сообщение
- **Excel-стиль**:
  - Строка формул для ввода/редактирования выражений
  - Синхронизация строки с current_expression через сигналы
  - Нажатие Enter вычисляет выражение
- **Безопасность**:
  - Валидация AST (запрещены вызовы функций, атрибуты, имена)
  - Потокобезопасность голосового ввода (threading.Lock)
  - Флаг should_stop для прерывания распознавания
