## 📖 О проекте

**Algorithmic Classifier v3** — это эксперимент по созданию классификатора изображений и текста **без использования нейронных сетей**, только на основе классических алгоритмов машинного обучения и эвристик.

### 🎯 Идеология

| Нейросети | Наш подход |
|-----------|------------|
| Чёрный ящик | Прозрачные правила |
| Требуют GPU | Работают на CPU |
| Миллионы параметров | Тысячи правил |
| Градиентный спуск | PairPatternNet (поиск пар) |
| 99.2% на MNIST | **82% на MNIST** |

> **Почему это важно?**  
> Нейросети дают 99.2%, но вы не знаете, ПОЧЕМУ цифра 7 — это 7.  
> Наш классификатор выдаёт понятные правила:  
> *"Если в 3-й строке серия из 5 чёрных пикселей И средняя яркость верхней половины < 0.3 → это 1"*

### 📊 Текущие достижения

| Задача | Метод | Точность |
|--------|-------|----------|
| **MNIST** | Супер-гибрид (5 модулей) | **82%** |
| **Логические числа** | PairPatternNet | ~100% |
| **Генерация текста** | 7 методов + оркестр | 🚧 в разработке |

### Coarse-to-Fine (почему только для оригинального)

```python
# Обучаем отдельные модели на разных масштабах размытия
scales = [
    ('ultra_blur', σ=3.0),  # самые грубые формы
    ('high_blur',  σ=2.0),
    ('med_blur',   σ=1.0),
    ('sharp',      σ=0.0),  # детали
]

# Уверенные предсказания на грубых масштабах
# Остальные уточняются на более чётких
```

### Почему рекурсия отключена (max_levels=1)

```
Уровень 1: 8000 картинок → 69% → 2481 ошибка
                                    ↓
Уровень 2: 2481 "сложных" картинок → обучается
                                    ↓
Проблема: ❌ Переобучение (мало данных, много правил)
Вывод:   ✅ Рекурсия НЕ РАБОТАЕТ → отключена
```

### Тест 3 (MNIST) — подробно

```python
from models.super_hybrid import SuperHybridNet
from utils import load_mnist

# Загрузка
X_train, y_train, X_test, y_test = load_mnist(10000, 2000)

# Создание супер-гибрида
model = SuperHybridNet(num_classes=10, confidence_threshold=0.15)

# Добавление модулей
model._add_ctf_module("Оригинальный (CTF)", extract_original_features)
model._add_ppn_module("Координатный", extract_coordinate_features)
model._add_ppn_module("CNN-подобный", extract_cnn_features)
model._add_ppn_module("Nonogram", extract_nonogram_features)
model._add_edit_module("EditDistance")

# Обучение
model.fit(X_train, y_train, max_pairs=300000, min_support=3)

# Предсказание
y_pred = model.predict(X_test)
acc = model.score(X_test, y_test)  # ~82%
```

## 🔧 Решение проблем

> [ERR-V3-001] - Низкая точность MNIST (<70%)

### Присутствует в версиях: [v3.0.0]
### Симптомы: Тест 3 показывает <70% вместо ожидаемых 82%
### Причина: Неверные параметры max_pairs или min_support
### Решение:
```python
# В super_hybrid.py увеличьте max_pairs
model.fit(X_train, y_train, max_pairs=500000, min_support=5)
```

---

> [ERR-V3-002] - Рекурсия вызывает переобучение

### Присутствует в версиях: [v3.0.0]
### Симптомы: Точность падает на валидации после 2-го уровня
### Причина: Мало данных на уровне 2
### Решение: В `hybrid_recursive.py` установите `max_levels=1`

---

> [ERR-V3-003] - Долгое обучение (>10 минут)

### Присутствует в версиях: [v3.0.0]
### Симптомы: PairPatternNet считает пары слишком долго
### Причина: Слишком много признаков (543) и пар
### Решение:
```python
# Уменьшите max_pairs
model.fit(X_train, y_train, max_pairs=100000, min_support=5)
```
