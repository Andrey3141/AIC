## 🧮 О проекте

Калькулятор с интерфейсом в стиле Excel (сетка кнопок + строка формул) и возможностью голосового ввода на русском языке. Работает полностью оффлайн благодаря интеграции с Vosk.

### Ключевые особенности

| Особенность | Описание |
|-------------|----------|
| **Excel-стиль** | Строка формул для ввода выражений + сетка кнопок |
| **Голосовой ввод** | Распознавание речи на русском языке оффлайн |
| **MVVM архитектура** | Строгое разделение логики, интерфейса и данных |
| **Безопасные вычисления** | AST-валидация без использования eval() |
| **Поддержка скобок** | Полноценные математические выражения |

---

## ✨ Возможности

- **Базовые операции**: `+` `-` `*` `/`
- **Скобки**: приоритет операций
- **Десятичные дроби**: через точку или запятую
- **Голосовой ввод**:
  - Числа прописью до 100 (например: "двадцать пять")
  - Операторы: "плюс", "минус", "умножить", "разделить"
  - Скобки: "открыть скобку", "закрыть скобку"
  - Точка/запятая для десятичных дробей
- **Обработка ошибок**:
  - Деление на ноль → сообщение
  - Нераспознанная команда → "Не распознано"
  - Синтаксическая ошибка → сообщение
- **Excel-стиль**:
  - Строка формул для ввода/редактирования
  - Нажатие Enter вычисляет выражение
  - Кнопка "=" для вычисления

# FAQ.md

> [ERROR-1] - Ошибка загрузки модели Vosk

### Присутствует в версиях: [v1.0.0] - текущая
### Симптомы: При запуске появляется сообщение "Предупреждение: Модель Vosk не найдена", голосовой ввод не работает
### Причина: Модель Vosk не скачана или распакована не в ту папку
### Решение: 
1. Скачайте модель: `wget https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip`
2. Распакуйте: `unzip vosk-model-small-ru-0.22.zip`
3. Создайте папку `model` в корне проекта
4. Переместите: `mv vosk-model-small-ru-0.22 model/`

---

### Присутствует в версиях: [v1.0.0] - текущая
### Симптомы: `pip install pyaudio` завершается с ошибкой компиляции
### Причина: Отсутствуют системные зависимости portaudio
### Решение (Ubuntu/Debian):

[ERROR-3] - Микрофон не работает / голос не распознаётся
Присутствует в версиях: [v1.0.0] - текущая
Симптомы: При нажатии кнопки "Голос" ничего не происходит или всегда "Не распознано"
Причина: Микрофон не выбран по умолчанию или низкая громкость
Решение:
Проверьте микрофон:
arecord -l
(должен показывать устройство)
Проверьте уровень звука:
arecord -d 5 test.wav && aplay test.wav
Увеличьте громкость микрофона в настройках системы
В коде можно уменьшить порог чувствительности (строка
if max_amplitude < 500:
в
voice_input_handler.py
)
[ERROR-4] - Голосовая команда не распознаётся правильно
Присутствует в версиях: [v1.0.0] - текущая
Симптомы: Сказали "пять плюс три", а в строке появляется "5 плюс 3" или "пять+три" в неправильном формате
Причина: Vosk распознал слова с ошибками или акцент
Решение:
Говорите чётко, с паузами между словами
Используйте простые команды: "пять плюс три", "двадцать разделить на четыре"
Сложные выражения разбивайте на несколько голосовых вводов
Можно вручную отредактировать строку формул
[ERROR-5] - Вычисление выдаёт "Ошибка в выражении"
Присутствует в версиях: [v1.0.0] - текущая
Симптомы: При нажатии "=" или Enter появляется сообщение об ошибке
Причина: Синтаксическая ошибка (например, два оператора подряд, незакрытые скобки)
Решение:
Проверьте строку формул на правильность
Пример правильного выражения:
(5+3)*2
Пример неправильного:
5++3
или
(5+3
Нажмите "C" для очистки и начните заново
[ERROR-6] - Деление на ноль выдаёт ошибку
Присутствует в версиях: [v1.0.0] - текущая
Симптомы: При делении на ноль появляется сообщение "Деление на ноль"
Причина: Математическая ошибка (деление на ноль)
Решение:
Это нормальное поведение калькулятора
Нажмите "C" для очистки
Введите корректное выражение (делите на ненулевое число)
[ERROR-7] - Калькулятор не запускается на Windows
Присутствует в версиях: [v1.0.0] - текущая
Симптомы: Ошибки при импорте PySide6 или pyaudio
Причина: Проект разработан для Linux, но может работать на Windows с доработками
Решение:
Установите PySide6:
pip install PySide6
Для pyaudio на Windows:
pip install pipwin
→
pipwin install pyaudio
Для Vosk: модель работает на любой платформе
Рекомендуется использовать WSL2 для полной совместимости
